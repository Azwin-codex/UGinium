# 📖 About UGinium

UGinium is a next-generation university collaboration platform designed to help students discover talent, build meaningful teams, showcase skills, and collaborate on innovative projects.

Instead of traditional social networking, UGinium focuses on **professional growth, project collaboration, and skill discovery** inside universities.

Whether you're searching for a developer, designer, AI engineer, researcher, or entrepreneur, UGinium helps students connect based on their abilities rather than just their profiles.

---

# ✨ Vision

Our vision is to build the operating system for university innovation.

# 🚀 Getting Started

Clone the repository

```bash
git clone git@github.com:Azwin-codex/UGinium.git
```

Move into the project

```bash
cd UGinium/apps/web
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
