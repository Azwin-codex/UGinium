# UGinium Academy Theme System v1.0

## Introduction

UGinium is a multi-campus digital academy.

Every university is represented as an individual Academy Campus while remaining part of the larger UGinium ecosystem.

The purpose of this system is to provide every university with its own visual identity without sacrificing the global UGinium brand.

The Academy Theme System separates what belongs to UGinium from what belongs to each Campus.

---

# Design Hierarchy

The visual system is divided into three immutable layers.

```
Layer 1
UGinium Core

↓

Layer 2
Campus Theme

↓

Layer 3
Student Identity
```

Every visual element must belong to one of these layers.

---

# Layer 1 — UGinium Core

The UGinium Core is permanent.

It defines the identity of the platform.

Nothing inside this layer may change between universities.

# Layer 3 — Student Identity

Students personalize the Academy through achievement rather than manual customization.

Identity is earned.

Never chosen.

Students unlock cosmetic upgrades by contributing to the Academy.

Examples

Titanium Identity Frame

Elite Module Finish

Founder Crest

Innovation Crest

Research Laureate

Guild Commander Border

Legendary XP Animation

Master Mentor Emblem

These cosmetics communicate prestige.

Not decoration.

---

# Progression Cosmetics

Every cosmetic must represent an achievement.

Cosmetics may never be purchased.

Prestige is earned.

Examples

Complete Major Projects

Lead Successful Guilds

Hackathon Victories

Research Publications

Community Mentorship

Verified Contributions

Innovation Awards

Exceptional Collaboration

The interface should visually tell the student's story.

# Recognition Rule

A screenshot from any Campus should immediately satisfy two conditions.

Condition One

The viewer instantly recognizes the university.

Condition Two

The viewer instantly recognizes UGinium.

If either identity disappears, the design has failed.

Both identities must coexist in harmony.

---

# Final Principle

UGinium is not a collection of university websites.

It is one Academy with many Campuses.

Every Campus is unique.

Every Campus belongs.

Every Campus feels unmistakably like UGinium.
